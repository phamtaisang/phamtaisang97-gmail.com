Tài liệu hướng dẫn phát triển Theme cho nền tảng bán hàng online - Onshop
    • Quy trình đăng ký/kích hoạt

      1. Liên hệ/Ký hợp đồng trở thành đối tác với Onshop

      2. Onshop tạo tài khoản công cụ phát triển theme cho đối tác

      3. Onshop gửi tài khoản công cụ phát triển theme cho đối tác

      4. Đối tác phát triển theme trên công cụ phát triển Onshop cung cấp

      5. Đối tác gửi bản zip Theme đã hoàn thiện cho Onshop

      6. Onshop kiểm tra, đánh giá theme đối tác đã gửi

      7. Nếu chưa đạt yêu cầu: Đối tác tiếp tục phát triển theme trên công cụ phát triển Onshop cung cấp, lặp lại bước 4-5-6

      8. Nếu đạt yêu cầu: Onshop thông báo cho đối tác và đẩy theme lên kho theme của Onshop

      9. Thực thi luồng thanh toán, hợp tác, lợi nhuận... giữa đối tác và Onshop

    1. Hướng dẫn sử dụng công cụ phát triển theme

      1. Tại thanh điều hướng bên trái: chọn Website > Giao diện

      2. Tại màn hình giao diện: bấm vào nút "Phát triển giao diện"

      3. Tại màn hình phát triển giao diện có các tính năng sau:

      *3.1 Danh sách giao diện đang phát triển, gồm:

      • Ảnh

        + Ảnh giao diện, yêu cầu có trong theme

      • Tên giao diện

        + Tên giao diện, lấy theo tên theme từ file .zip

      • Hành động

        + Sửa giao diện Đến trang sửa giao diện

        + Xoá giao diện Xoá giao diện đang phát triển

        + Tải giao diện về Tải giao diện về

      • [link] Sử dụng

        + Sử dụng giao diện hiện tại là giao diện cho trang cửa hàng bên ngoài

      *3.2 Nút: tải giao diện lên

      • Tải giao diện lên. Quy định:

        - Loại file: .zip

        - Thư mục bên trong: tên theme > các thư mục con theo tiêu chuẩn cấu trúc bộ theme

#default.json

File định nghĩa màu sắc và font chữ mặc định cho theme
Các nội dung có thể sửa được trong file default.json:
"config_theme_color" > "main" > "main" > "hex": "#fda7a8": định nghĩa màu chủ đạo, tức là các màu có thể thay đổi theo cấu hình "màu chủ đạo" ở phần thiết lập theme builder bởi admin
"config_theme_color" > "main" > "button" > "hex": "#fda7a8": định nghĩa màu chủ đạo, tức là màu các nút bấm có thể thay đổi theo cấu hình "màu nút bấm" ở phần thiết lập theme builder bởi admin
"config_theme_color" > "all" > "font": "Tahoma": định nghĩa font chữ được sử dụng, tức là font chữ có thể thay đổi theo cấu hình "font chữ" ở phần thiết lập theme builder bởi admin
"config_theme_color" > "all" > "font": "supported-fonts": định nghĩa font chữ có thể được lựa chọn khi cấu hình "font chữ" ở phần thiết lập theme builder bởi admin

Ví dụ

                    
                    {
                        "config_theme_color": {
                            "main": {
                                "main": {
                                    "hex": "#fda7a8", "visible": 1
                                }
                                ,
                                "button": {
                                    "hex": "#FDA7A8", "visible": 1
                                }
                            }
                        }
                        ,
                        "config_theme_text": {
                            "all": {
                                "title": "Font Tahoma",
                                "font": "Tahoma",
                                "supported-fonts": [
                                    "Open Sans",
                                    "IBM Plex Sans",
                                    "Courier New",
                                    "Roboto",
                                    "Nunito",
                                    "Arial",
                                    "DejaVu Sans",
                                    "Tahoma",
                                    "Time News Roman"
                                    ]
                            }
                        }
                    }

Trang quản lý tài khoản

 

Chi tiết API

 

?route=account/account

 

Dữ liệu trả về :

- breadcrumbs


                    "array( [ 'text' => '
                    ', 'href' => 'common/home' ], [ 'text' => '
                    ', 'href' => 'checkout/profile' ] ); Hiển thị text:
                    link tương ứng: common/home / checkout/profile"

- success

Text thông báo thành công

- edit

Link edit tài khoản: account/edit

- password

Link đổi mật khẩu tài khoản: account/password

- address

Link sửa địa chỉ tài khoản: account/address

- order

Link danh sách đơn hàng của tài khoản: account/order
